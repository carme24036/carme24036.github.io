RoboDog

Over the past few months, I, as well as some other students, have been working on the class' Robot Dogs. We named the project RoboDog. Recently, however, I have taken over the development of the dogs.

  • The original idea was to program an Alexa speaker to use ChatGPT to give the dogs commands, but we realized that the dog has its own speakers and microphone and no longer need Alexa. Now, the goal is to use the dogs' built-in speakers and microphone to create a speech to text program in which the dog can interpret words and turn that into code that it can run.

  • So far, I have created a program that uses an OpenAi API to have a user input information and get a response from the API, similar to ChatGPT. I am using the program to give the RoboDog commands. However, this is only the first step in the development of the dog. The next step in the development of this project is to figure out how to get access to the microphone and speaker of the dog so it can be linked with a speech-to-text/text-to-speech program.
